# Clicky

Clicky is used in some of our videos. Written in KickAssembler for the Commodore 64.
The eyes are sprites.

1-0, q-p (Change Eyes)

a-l (Change Mouth)

F1 - Toggle Scar

F2 - Toggle Crying

F7 - Change Colors
